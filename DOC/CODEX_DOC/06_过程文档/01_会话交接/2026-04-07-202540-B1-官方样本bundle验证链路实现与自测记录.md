## 3. 关键实现决策

1. 选择 `spine-runtimes@4.2` 作为官方样本真源
2. 玩家样本固定为 `spineboy`
3. 敌人样本固定为 `raptor`
4. 使用官方导出的 `.json` skeleton，而不是 `.skel`
   - 原因：当前阶段可直接提取 `animations / slots / defaultSkin`
5. 正式 bundle 内统一折叠为稳定文件名：
   - `spineboy.json / spineboy.atlas / spineboy.png`
   - `raptor.json / raptor.atlas / raptor.png`
6. 下载器实现了 `fetch -> curl` 回退
   - 原因：本机 `curl` 可通过代理访问 GitHub raw，而 Node 内置 `fetch` 不能稳定走该代理
