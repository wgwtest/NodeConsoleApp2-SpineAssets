### 3.2 契约文件与样例自洽性检查

执行命令：

```bash
node --input-type=module -e '
import fs from "node:fs";
const catalog = JSON.parse(fs.readFileSync("spec/examples/presentation_catalog.example.json", "utf8"));
const bundle = JSON.parse(fs.readFileSync("spec/examples/bundle_manifest.example.json", "utf8"));
const character = JSON.parse(fs.readFileSync("spec/examples/character_manifest.example.json", "utf8"));
if (catalog.meta.schemaVersion !== bundle.sourceCatalog.schemaVersion) throw new Error("sourceCatalog schemaVersion mismatch");
if (!bundle.characters.some((item) => item.presentationId === character.presentationId)) throw new Error("character manifest not referenced by bundle");
if (character.skeletonFile !== `${character.presentationId}.skel`) throw new Error("skeleton filename mismatch");
if (character.atlasFile !== `${character.presentationId}.atlas`) throw new Error("atlas filename mismatch");
console.log(`CONTRACT OK catalog=${catalog.meta.schemaVersion} bundle=${bundle.schemaVersion} character=${character.schemaVersion}`);
'
```

结果：

1. 输出 `CONTRACT OK ...`
2. `presentation_catalog -> bundle_manifest -> character_manifest` 的最小引用链成立
3. 角色资源命名与 `presentationId` 对齐，没有出现样例自相矛盾
