# B1 Official Sample Bundle Validation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a repeatable `B1` workflow that fetches official `Spineboy` and `Raptor` export samples, assembles a contract-compliant sample bundle, validates it, and generates a static preview page with evidence outputs.

**Architecture:** Keep the workflow split into four CLI stages: sample metadata, fetch, build, validate, and preview. Treat official sample files as input fixtures under `workspace/samples/official`, generate a formal bundle under `workspace/exports/b1_official_samples`, and keep all validation and preview logic reading only the bundle output so the later `C1` consumer path stays aligned with the same contract.

**Tech Stack:** Node.js ESM, `node:test`, filesystem-based tooling, existing JSON schema contracts, static HTML generation

export async function fetchOfficialSamples({ outputRoot, downloadBinary, sampleCatalog = OFFICIAL_SAMPLE_CATALOG }) {
  for (const sample of sampleCatalog) {
    const sampleDir = path.join(outputRoot, sample.presentationId);
    await fs.mkdir(sampleDir, { recursive: true });
    for (const asset of sample.assets) {
      const bytes = await downloadBinary(asset.url);
      await fs.writeFile(path.join(sampleDir, asset.outputName), bytes);
    }
  }
}
```

export async function validateSampleBundle({ bundleRoot }) {
  const bundle = await readJson(path.join(bundleRoot, 'bundle_manifest.json'));
  for (const item of bundle.characters) {
    const manifestPath = path.join(bundleRoot, item.characterManifest);
    const manifest = await readJson(manifestPath);
    await assertFile(path.join(path.dirname(manifestPath), manifest.skeletonFile));
    await assertFile(path.join(path.dirname(manifestPath), manifest.atlasFile));
  }
  return { ok: true, checkedCharacters: bundle.characters.length };
}
```
